# java-cli-maven-surefire-jacoco-pmd-jbehave-testng-hello-world

## Description
A POC for maven app using testng
and jbehave framework with pmd,
jacoco and surefire plugin.

## Tech stack
- java
- maven
  - testng
  - surefire
  - jacoco
  - pmd
  - jbehave

## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
